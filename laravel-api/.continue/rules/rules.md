# Laravel Coding Conventions

## General

*   Follow the PSR-12 coding standard.
*   Use descriptive variable and function names.
*   Add comments to complex logic.

## Models

*   Model files should be placed in the `app/Models` directory.
*   Use Eloquent ORM for database interactions.
*   Define relationships between models.

## Controllers

*   Controller files should be placed in the `app/Http/Controllers` directory.
*   Use resource controllers for CRUD operations.
*   Use dependency injection for dependencies.

## Routes

*   Define API routes in the `routes/api.php` file.
*   Use route groups for versioning and authentication.
